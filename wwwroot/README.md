# wwwroot Klasör Yapısı

Bu klasör, uygulamanın statik dosyalarını içerir.

## Görsel Dosyaları

Görsel dosyaları için standart yapı:

- `/img/posts/` - Blog yazılarının görselleri
  - `default.jpg` - Görseli olmayan yazılar için varsayılan resim
- `/img/profiles/` - Kullanıcı profil resimleri
  - `default.jpg` - Profil resmi olmayan kullanıcılar için varsayılan resim

Tüm resim yüklemeleri bu dizinlere yapılmaktadır.
